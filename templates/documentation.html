Getting Started

This guide will walk you through the basics of using B.Tech Hub. You’ll learn how to navigate through the dashboard, access resources, calculate SGPA, and more.

Features Overview

  • University Info – Access AKTU regulations, ordinances, and grading systems.
  • Syllabus Hub – View semester-wise and branch-wise curriculum.
  • SGPA Calculator – Calculate your SGPA using AKTU's grading scale.
  • Resources – Access academic materials, tools, and guides.

Technical Information

The platform is built using Flask (Python) for the backend, Tailwind CSS for the frontend, and SQLite for data storage. It is designed for responsiveness and usability across devices.
